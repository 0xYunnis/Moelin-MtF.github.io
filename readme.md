## 项目介绍
这里是TransGuide，是一个或许可以给部分跨性别群体提供导航的地方。

跨性别（跨儿）：跨性别女性（MtF）、跨性别男性（FtM）、非二院/性别酷儿、跨性者、跨装者、间性人等性别多元群体。
